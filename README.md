Spend Habit

Native iOS application developed with SwiftUI and SwiftData for personal expense tracking.

Features:
- Expense management
- Categories
- Analytics
- Charts
- Authentication
- Cloud synchronization
- REST API integration

Technologies:
- SwiftUI
- SwiftData
- Async/Await
- MVVM
- Spring Boot
- PostgreSQL
- Docker
